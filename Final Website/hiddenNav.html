Hidden Nav

Hidden Navbar

The dashed line represents the webpage

This kind of hidden nav does not use any JavaScript so it is slightly hacky. It uses a hidden checkbox linked to the MENU/CLOSE button to change the opacity of the "hidden" menu along with a transform to get that translation that occurs.


The code below does not accurately represent the above hidden nav for the purposes of displaying an example, but it will replicate the hidden nav I use for this website!

Hidden Navbar » HTML

<nav>
   <input type="checkbox" id="checkboxNav"/>
   <ul class="sidenav">
      <label for="beginCheckboxSide"><a href="#"><b>Beginner</b></a></label>
      <label for="advanceCheckboxSide"><a href="#"><b>Advanced</b></a></label>
      <label for="expertCheckboxSide"><a href="#"><b>Expert</b></a></label>
      <label for="aboutCheckboxSide"><a href="#"><b>About Me</b></a></label>
   </ul>
   <section>
      <label id="menu" for="checkboxNav">Menu</label>
      <label id="close" for="checkboxNav">Close</label>
   </section>
</nav>
