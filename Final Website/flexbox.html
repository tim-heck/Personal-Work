Flex-box

Flex-box is an amazing styling method. It creates an nice organized interface that is fully responsive. It works by creating a container, applying flex-box to that container, and filling it will items that will adopt the flex-box styling. It has many different styling techniques that can be applied to the items within the container with flex-box applied to it. If you are on a desktop, adjust the width of your browser to see how flex-box reacts!
