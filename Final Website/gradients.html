Gradients

Gradients are different than the other styles when it comes to completing a website. Gradients are not essential and in my opinion, are not used very much and not as appealing as other types of color styling. Yet, if an ideal opportunity presents itself now you will know how to create them!

Vertical Gradient

Vertical Gradient » HTML

<section id="gradient1"></section>

Vertical Gradient » CSS

#gradient1 {
   height: 200px;
   background: #c2986b; /* For browsers that do not support gradients */
   background: -webkit-linear-gradient(#c2986b, #000000); /* For Safari */
   background: -o-linear-gradient(#c2986b, #000000); /* For Opera */
   background: -moz-linear-gradient(#c2986b, #000000); /* For Firefox */
   background: linear-gradient(#c2986b, #000000); /* Standard syntax (must be last) */
}
