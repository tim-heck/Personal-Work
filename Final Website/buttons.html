Buttons

Buttons are pretty standard when it comes to a website. In these examples anchor tags, or a tags, which are normally used for links, are used for these buttons. Adding padding and a background color gives the look of a button and is becoming more common than the button tag.

CSS Applied to All Buttons

.btn {
   background-color: #ffffff;
   border-radius: 4px;
   color: #000000;
   padding: .5em 2em;
   text-align: center;
   text-decoration: none;
   display: inline-block;
   font-size: 1em;
   cursor: pointer;
   margin-top: 16px;
}
