Hover

*Note: Transitions are being used! Go to the Advanced > Transition to learn more!

The hover effect is very similar to the click effect, only it happens when you hover over the element with the hover effect. Again, mostly it's a background or text color change but sometimes it's a shadow added or a translation. It can also be applied to more elements than just links or buttons.

CSS Applied to All Buttons

.btn {
   background-color: #ffffff;
   border-radius: 4px;
   color: #000000;
   padding: .5em 2em;
   text-align: center;
   text-decoration: none;
   display: inline-block;
   font-size: 1em;
   cursor: pointer;
   margin-top: 16px;
}

CSS Applied to All Links

a {
   text-decoration: none;
   color: #000000;
}

Buttons

*Note: The '!important' overrides any CSS that was prevously applied to the element.

Hover over me! Hover over me! Hover over me!

Buttons » HTML

<a class="btn btn1" href="#">Hover over me!</a>
<a class="btn btn2" href="#">Hover over me!</a>
<a class="btn btn3" href="#">Hover over me!</a>

Buttons » CSS

.btn1 {
   /*uses CSS for all buttons*/
}

.btn1:hover {
   text-decoration: underline;
   background-color: green!important;
   color: #ffffff!important;
}

.btn2 {
   /*uses CSS for all buttons*/
}

.btn2:hover {
   font-family: Open Sans, sans-serif!important;
   background-color: #000000!important;
   color: yellow!important;
}

.btn3 {
   /*uses CSS for all buttons*/
}

.btn3:hover {
   padding: 2em 2.5em;
   transition: .5s; /*Duration of the changing size*/
}

.div1 {

   margin: 0 0 1em 0;
   background-color: #ffffff;

}

.div1:hover {
   padding: 1.5em;
   margin-right: 75%;
   background-color: blue;
   color: #ffffff;
   transition: .8s;
}

There are many things you can do with hover in CSS. It can be applied to almost any type of element. Experiment with ":hover" with elements on your website and find all of the different possibilites that are available!
