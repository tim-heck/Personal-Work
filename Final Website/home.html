Home

This website was built to help any and all users that know little or a lot about HTML or CSS. It is a tool for you to use! Each page will have one or more examples about the CSS styling that the page is representing. After each example there will be an HTML section and a CSS section with the HTML code used to make that example and the CSS code used to style it. There may also be a CSS section of code that is applied to all examples at the top of the page before any examples are shown.

Take your time to look at the example and the code. Once you feel you have a grasp on what the CSS is doing, you can simply copy and paste the HTML and the CSS into your code! Depending on what is already in your HTML or CSS files you may have to make some modifications to the copied code to fit to your website. This website was also built for desktop use. Many of the styling methods will not work the same as on a desktop, or they will not work the same. You will be able to see most, if not all, of the content on a mobile phone, but it will not be the same experience. So get on a desktop! Now you can start by choosing a styling method from the dropdown menus above!
