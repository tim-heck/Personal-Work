Click

The click effect happens when you click on some element. It is usually used with a link or a button, it can be applied to almost any element though. A text color or background color is most likely changed on a click, but other things can be changed to. Experiment with it!s

CSS Applied to All Buttons

.btn {
   background-color: #ffffff;
   border-radius: 4px;
   color: #000000;
   padding: .5em 2em;
   text-align: center;
   text-decoration: none;
   display: inline-block;
   font-size: 1em;
   cursor: pointer;
   margin-top: 16px;
}

CSS Applied to All Links

a {
   text-decoration: none;
   color: #000000;
}

Click for a Button

Click Me! Click Me!

Click Button » HTML

<a class="btn btn1" href="#">Click Me!</a>
<a class="btn btn2" href="#">Click Me!</a>

Click Button » CSS

.btn1 {
   /*uses CSS for all buttons*/
}

.btn1:active {
   background-color: purple;
   color: #ffffff;
}

.btn2 {
   /*uses CSS for all buttons*/
}

.btn2:active {
   background-color: blue;
   color: red;
}
