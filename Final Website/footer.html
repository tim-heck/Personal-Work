Footer

The footer is also an essential piece to a website. Like the navbar, it's something that completes your site. The experience of the user is what is key to a website, the footer will allow the user to know where the bottom of the page is as well as containing information that the user may need to see.

Footer

The code below does not accurately represent the above footer for the purposes of displaying an example, but it does represent the footer used on this website so it will work!

Footer » HTML

/*Your footer tag should be put at the bottom of your html document, but still inside the body tag*/
<footer>
   <h6 class="footer">
      <a href="../Final%20Website/about.html">
      Contact Me</a> | <a href="https://
      www.instagram.com/timtheheck/">
      Follow Me on Instagram</a>
   </h6>
</footer>

Footer » CSS

h6.footer {
   width: auto;
   padding: 1.5em 0em 1.5em 0em;
   font-size: 0.75em
}

footer {
   display: flex;
   flex-direction: row;
   flex-wrap: wrap;
   justify-content: center;
   background-color: #ffffff;
   width: 100%;
   position: relative; /*The position can vary depending on the webpage*/
   right: 0;
   bottom: 0;
   left: 0;
   height: 3em;
}

For the position of the footer it depends on the content of the website. The 'relative' position will put it at the bottom of your content that is within the natural flow of the webpage. When an element has a position of 'absolute' it is taken out of the natural flow of the webpage, so your footer will display above or behind 'absolute' elements. If you do not have enough content on your webpage to push the footer to the bottom, use 'absolute' for positioning your footer. If you want your footer to constantly be seen on the webpage, use 'fixed' for your position.
