Shadow

A shadow effect applied to the right element can be really cool. You can create the illusion of a floating object and adding more interest to your website. It can be applied with a click or hover effect, or just by itself permently on an element.

CSS Applied to each Shadow Text

.shadowText {
   font-size: 2em;
   display: inline;
   margin-right: .5em;
   white-space: nowrap;
}

CSS Applied to each Shadow Box

.shadowBox {
   width: 10em;
   height: 6em;
   background-color: #c2986b;
}

.shadowBox > p {
   padding: 1em;
}
