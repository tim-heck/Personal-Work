Transition

Transitions are not essential to your website either, but they can add a lot of character and interest. A simple transition can go a long way and not having too many is also a good way to go. If there was a lot, they would be destracting for a user which will most likely make them leave and not come back. They can be applied on a hover effect and can be used as a time duration for different translations or effects.

CSS Applied to All Transitions

.transition {
   width: 10em;
   height: 6em;
   color: #ffffff;
   background: #000000;
   -webkit-transition: width 1.5s; /* Duration of transition: For Safari */
   transition: width 1.5s; /* Duration of transition */
}

.transition:hover {
   width: 30em;
}

Transitions » CSS

/* For Safari */
#transition1 {-webkit-transition-timing-function: linear;}
#transition2 {-webkit-transition-timing-function: ease;}
#transition3 {-webkit-transition-timing-function: ease-in;}
#transition4 {-webkit-transition-timing-function: ease-out;}
#transition5 {-webkit-transition-timing-function: ease-in-out;}

/* Standard syntax */
#transition1 {transition-timing-function: linear;}
#transition2 {transition-timing-function: ease;}
#transition3 {transition-timing-function: ease-in;}
#transition4 {transition-timing-function: ease-out;}
#transition5 {transition-timing-function: ease-in-out;}
