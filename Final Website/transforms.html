Transforms

Transforms, like transitions and animations, are simply styling that can only add to your website making it more interesting, but not essential.

CSS Applied to All Transforms

*Note: Some transforms will not work on Internet Explorer 9 or earlier versions.

.transform {
   width: 100%;
   height: 7em;
   background-color: #000000;
   color: #c2986b;
   transition: 1s;
}

.transform > p {
   padding: 1em;
}

Rotations » CSS

#transform1:hover {
   -webkit-transform: rotateX(180deg); /* Safari */
   transform: rotateX(180deg); /* Standard syntax */
}

#transform2:hover {
   -webkit-transform: rotateY(180deg); /* Safari */
   transform: rotateY(180deg); /* Standard syntax */
}

#transform3:hover {
   -webkit-transform: rotateZ(180deg); /* Safari */
   transform: rotateZ(180deg); /* Standard syntax */
}
