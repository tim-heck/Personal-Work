Hover Dropdown Menu

A hoverable dropdown menu is the most common solution for dropdown menus without using JavaScript. There are ways to have a clickable dropdown menu without JavaScript, but it's very complicated. The only problem with this type of dropdown menu is when on a mobile device a user has to click to open the menu and then it cannot be closed.

Hover Dropdown » CSS

#navbar1 {
   padding: 0;
   background-color: #ffffff;
}

.navUl3 {
   list-style-type: none;
   margin: 0;
   padding: 0;
   overflow: hidden;
   background-color: #ffffff;
}

.navLi3 {
   float: left;
}

.navLi3 a, .dropbtn {
   display: inline-block;
   color: #000000;
   text-align: center;
   padding: 14px 16px;
   text-decoration: none;
}

.navLi3 a:hover, .hoverDropdown:hover .dropbtn {
   background-color: #000000;
   color: #c2986b;
}

.navLi3.hoverDropdown {
   display: inline-block;
}

.dropdown-content {
   display: none;
   position: absolute;
   background-color: #ffffff;
   min-width: 160px;
   box-shadow: 0px 8px 16px 0px rgba(0,0,0,0.2);
}

.dropdown-content a {
   color: black;
   padding: 12px 16px;
   text-decoration: none;
   display: block;
   text-align: left;
}

.hoverDropdown:hover .dropdown-content {
   display: block;
}
